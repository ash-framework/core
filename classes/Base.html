Defines mixins for class.

Gets passed a register function that can be called to mix mixin class methods in.

Call register multiple times to mixin multiple mixins.

class MyMixin1 {
    method1 () {}
  }
  
  class MyMixin2 {
    method2 () {}
  }
  
  class MyClass extends Ash.Base {
    static mixins(register) {
      register(MyMixin1)
      register(MyMixin2)
    }
  }

Instantiating MyClass will result in an object with methods method1 and method2 mixed in.

Parameters:

  • register Function
    • mixin registration function. Can be called multiple times to register additional mixins.
